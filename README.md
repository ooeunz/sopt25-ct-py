### repository using rule
* 한주동안 푼 문제들은 각각의 branch로 commit 합니다.

* branch 만드는 방법
`git checkout -b 브랜치네임` // branch 생성 후 자동으로 해당 branch 변경됩니다.

* branch 네이밍
`week1_본인의깃헙아이디`    // 저로 예를들면 week1_ooeunz이 됩니다.

* Pull request 네이밍
`[신윤재] 1주차 : 1000, 10001`  // 숫자는 백준 문제번호 입니다.
